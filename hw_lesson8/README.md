# TS - lesson 8
Interface  →  defines shape/contract, no implementation, erased at runtime
Class      →  blueprint with implementation, compiled to JS
implements →  class satisfies interface contract
extends    →  class/interface inherits from another
abstract   →  cannot be instantiated, forces subclass implementation
public     →  accessible everywhere (default)
protected  →  accessible in class and subclasses
private    →  accessible only within the declaring class
readonly   →  can be set once, then immutable
static     →  belongs to the class, not instances
